ZRP Examples
_____________

The `ZRP User Guide <./examples/ZRP-User-Guide.ipynb>`_ provides an introduction and context to the ZRP and associated example notebooks provided in this folder. The `ZRP Tutorial <./examples/ZRP-Tutorial.ipynb>`_ is an example notebook demonstrating the use of the ZRP by running a sample set of predictions on New Jersey mayors. The `Alabama ZRP Predictions Tutorial <./examples/AL_ZRP_Predictions_Tutorial.ipynb>`_ is a notebook demonstrating ZRP validation on the state of Alabama.
